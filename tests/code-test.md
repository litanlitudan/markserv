# Code Block Test

This markdown file tests code blocks with line numbers.

## JavaScript Example

```javascript
function hello() {
  console.log('Hello, World!');
}

function add(a, b) {
  return a + b;
}

hello();
console.log(add(2, 3));
```

## Python Example

```python
def greet(name):
    print(f"Hello, {name}!")

def multiply(x, y):
    return x * y

greet("World")
result = multiply(5, 3)
print(result)
```
